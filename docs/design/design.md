# Design Guidelines

## 1. Brand Direction

This portfolio should feel modern, minimal, professional, and design-driven.

Core keywords:
- Modern
- Minimal
- High-end
- Clear hierarchy
- Strong visual rhythm
- Business-oriented

## 2. Visual Style

- Use large whitespace.
- Use clear typography hierarchy.
- Avoid decorative clutter.
- Keep the interface clean, calm, and premium.
- Use strong contrast for key CTA buttons and important numbers.

## 3. Layout Principles

- Each section should have one clear purpose.
- Important information should appear above the fold.
- Project cards should be easy to scan.
- Use consistent spacing between sections.
- Avoid overly dense text blocks.

## 4. Typography

- Chinese: PingFang SC / system sans-serif
- English: Inter / system sans-serif
- Headings should be strong and concise.
- Body text should be readable and not too small.

## 5. Components

Buttons:
- Clear CTA wording
- Obvious hover state
- Consistent radius and padding

Cards:
- Clean structure
- Clear title, description, and project value
- 16:9 thumbnail ratio preferred

Navigation:
- Simple
- Clear
- No unnecessary menu items

## 6. Interaction

- Transitions should be subtle and smooth.
- Hover effects should support understanding, not distract.
- Avoid exaggerated animation.
- Mobile experience must remain clear and usable.

## 7. Content Tone

The copy should be:
- Professional
- Concise
- Confident
- Business-aware
- Result-oriented
